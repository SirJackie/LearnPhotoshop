# Learn Photoshop

## 全屏、菜单栏

F: Full Screen

Tab: Hide Windows

Shift + Tab: Hide Right Side Windows

## Z：缩放工具

**Alt + 鼠标滚轮：快速缩放**

**Ctrl + 1：缩放到100%**

**Ctrl + 0：缩放到适合屏幕的大小**

缩放工具时Alt：切换放大缩小

缩放工具时点击or左右拖动鼠标：放大缩小

## H：抓手工具

**空格 + 鼠标左键：快速移动**

按住空格：抓手工具

## 两张图片对比查看

窗口=》排列=》双联

窗口=》排列=》匹配缩放

窗口=》排列=》匹配位置

H（抓手）=》滚动所有窗口

Z（缩放）=》缩放所有窗口

## R：旋转工具

注意，旋转工具仅改变查看方式，不改变文件

Esc：复位旋转

## 图层

图层=》新建=》图层背景

## E：橡皮擦工具

如果是普通图层：擦成透明

如果是背景图层：擦成白色
